FAQ - Application d'échange de devises

Question 1: Comment puis-je créer un compte?
Réponse 1: Pour créer un compte, allez sur la page d'inscription et suivez les instructions fournies. Remplissez le formulaire avec les informations nécessaires et cliquez sur "S'inscrire".
Question 2: Comment publier une demande d'échange de devises?
Réponse 2: Accédez à la section "Publier une demande" dans votre tableau de bord. Sélectionnez les devises que vous souhaitez échanger, spécifiez le montant et cliquez sur "Publier". Votre demande sera visible pour d'autres utilisateurs.
Question 3: Comment puis-je consulter mes offres?
Réponse 3: Rendez-vous sur la page "Mes offres" dans votre tableau de bord. Vous verrez une liste de vos offres d'échange actuelles avec les détails correspondants.
